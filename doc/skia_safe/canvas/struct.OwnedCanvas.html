#[repr(transparent)]pub struct OwnedCanvas<'lt>(_, _);

A type representing a canvas that is owned and dropped when it goes out of scope and is bound to the lifetime of another instance. Function resolvement is done via the Deref trait.

pub unsafe fn surface(&mut self) -> Option<Surface>[src]

Safety

This function is unsafe because it is not clear how exactly the lifetime of the canvas relates to surface returned. TODO: It might be possible to make this safe by returning a kind of reference to the Surface that can not be cloned and stays bound to the lifetime of canvas. But even then, the Surface might exist twice then, which is confusing, but probably safe, because the first instance is borrowed by the canvas. See also OwnedCanvas, Surface::canvas().
